Detailed Description

Expansion class of DRT_DRT for periodic BC in the vertical direction.

A r2r transformation is performed horizontally, i.e. along the lines of the matrices, and a discrete fourier transformation is performed vertically, i.e. along the columns.

Note
Do not copy or assign any Objects of this class!!

execute a c2r transposing transformation

First perform a linewise discrete fourier transform followed by a transposition and a linewise discrete real transform. The result is the same as if you had used the drt_dft class on the transposed input.

Parameters
inout_TNon void complex matrix of size (real_cols, real_rows/2 + 1) Content on output is the one of swap on input.
swapCan be void. Size has to be (real_rows, real_cols). Contains the solution on output.
