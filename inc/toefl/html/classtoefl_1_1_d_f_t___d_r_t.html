Detailed Description

Expansion class of DRT_DRT for periodic BC in the horizontal direction.

First transforms linewise r2c, then transposes (r2r) and transforms linewise r2r. The result is transposed back (r2r). The Backward transform goes the same way in the other direction.

Note
Because of the extra transposes the transformation is not as fast as it could be. The fastest would be a 2d r2r transform of r2hc type. But then the halfcomplex format is so ugly that we don't want to do this.
