pointwise multiply the n x n Matrix of coefficients by a n-vector of matrices

Compute the system m0 = c00*m0 + c01*m1 + c02*m2 + ..., m1 = ... where all of the elements are matrices and matrix-matrix multiplications are done pointwise.

Template Parameters
T1type of the coefficients i.e. double or std::complex<double>
Ttype of the matrix elements, i.e. double or std::complex<double>
Parameters
cthe coefficient matrix
inInput vector of matrices
outOutput vector of matrices. Contains solution on output. Multiplication is done inplace if in and out reference the same object!
